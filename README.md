# React Account Table

A React.js application for managing account data with filtering, pagination, Excel export, and responsive design.

---

## How to Run Locally

npm install  
npm run dev  

The application will start at:  
http://localhost:5173

---

## Responsive Design

- Desktop: Full table view with pagination  
- Mobile: Horizontally scrollable table for better usability
